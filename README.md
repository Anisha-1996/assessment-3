# Music Player -- React Machine Test

## Overview

A YouTube Music--style audio player built using **React (Vite)** and
**Material UI**.

The application allows users to upload audio files, edit song names,
toggle between table and grid views, and control playback using a
bottom-fixed music player.

------------------------------------------------------------------------

## Features

-   Upload audio files (audio-only validation)
-   Drag & drop support
-   Edit uploaded song names
-   Toggle between **Table View** and **Grid View**
-   Bottom fixed audio player
-   Play / Pause controls
-   Next / Previous navigation
-   Autoplay next song after finish
-   Timeline progress slider
-   Highlight currently playing song
-   Fully responsive design

------------------------------------------------------------------------

## Tech Stack

-   React (Vite)
-   Material UI (MUI)   

------------------------------------------------------------------------

## Installation

Install dependencies:

``` bash
npm install
```

Run the development server:

``` bash
npm run dev
```
